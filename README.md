# washwell_fire_official

워시웰 화재세탁 공식 홈페이지 저장소입니다.

## 목적

- 워시웰 화재세탁 브랜드 전용 홈페이지 관리
- 런드리박스 홈페이지와 별도 저장소로 분리 운영
- 화재세탁, 연기피해, 그을음, 분진피해, 화재보험 제출자료 안내 중심의 랜딩페이지 운영

## 주요 파일

- `index.html` : 홈페이지 본문
- `styles.css` : 디자인 스타일
- `script.js` : 모바일 메뉴, 요금표 검색 및 필터 기능

## 공개 예정 주소

GitHub Pages 설정 후 아래 주소로 공개됩니다.

## 운영 메모

- 최종 전화번호, 카카오톡 상담 링크, 워시웰 로고 원본 적용 여부 확인 필요
- 요금표 금액은 대표 품목 예시 기준이며 실제 견적은 품목 상태와 오염 정도에 따라 달라질 수 있음
